###  操作方法
・右矢印キー：右へ移動<br>
・左矢印キー：右へ移動<br>
・下矢印キー：下へ移動<br>
・上矢印キー：90度回転<br>


###  基本機能

- キーボードによるブロックの操作（左右・下・回転）
- 揃った列に応じたスコア加算
- 一時停止
- bgm,seの再生
- top scoreの保持（自己ベスト）

###  未実装、追加予定の機能

- ミノの落ちてくる場所の予測
- ミノのグラフィック向上（影や枠線で立体感を出す）
- welcome画面における背景の実装（現在はプレイ時のミノが見えてしまっている）
- 下移動、全消しで特殊スコア加算
- ハードドロップ
