##  [Tetris]()


###  操作方法
・右矢印キー：右へ移動<br>
・左矢印キー：右へ移動<br>
・下矢印キー：下へ移動<br>
・上矢印キー：90度回転<br>

###  基本機能

- キーボードによるブロックの操作（左右・下・回転）
- 揃った列に応じたスコア加算
- 一時停止
- bgm,seの再生
